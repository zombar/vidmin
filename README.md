## Features

- **Local Video Playback** - Play video files directly from your system with native performance
- **Progressive Downloads** - Download videos from YouTube and other platforms with real-time progress tracking
- **Smart Window Management** - Automatically resize window to match video dimensions while maintaining aspect ratio
- **Format Conversion** - Built-in FFmpeg integration for converting incompatible video formats
- **Drag & Drop** - Drop video files or URLs to instantly play or download
- **Modern Interface** - Clean, borderless design with intuitive controls powered by Vidstack

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Run tests
npm test

# Build for production
npm run build

# Package application
npm run package
```

## Tech Stack

- **Electron** - Cross-platform desktop framework
- **React** - UI components and state management
- **TypeScript** - Type-safe development
- **Vidstack** - Advanced video player library
- **yt-dlp** - Video download capabilities
- **FFmpeg** - Media conversion and processing
